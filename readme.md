
# 🎉 Sloganify – Générateur de Slogans IA

## 📌 Description  
Sloganify est une application web qui aide les entreprises, startups et freelances à trouver des slogans créatifs et percutants.  
L’utilisateur saisit le nom de son entreprise et une brève description de son activité, puis l’IA génère instantanément une liste de slogans accrocheurs et mémorables.  

## 🚀 Fonctionnalités
- Génération IA de slogans adaptés au secteur d’activité.  
- **Version gratuite** : jusqu’à **3 slogans/jour**.  
- **Abonnement Premium 👑** :  
  - Générations illimitées  
  - Sauvegarde des slogans favoris (en cours de développement)  
  - Expérience sans limites pour booster la créativité  

## 📂 Pages principales
- **Accueil** : Présentation de l’outil et mise en avant des avantages.  
- **Générateur** : Interface où l’utilisateur entre ses informations et reçoit ses slogans.  
- **Premium** : (⏳ à venir) Page dédiée à l’abonnement et aux fonctionnalités exclusives.  
- **Favoris** : (⏳ à venir) Page pour gérer et sauvegarder les slogans préférés.

# 🎉 Sloganify – AI Slogan Generator

## 📌 Description  
Sloganify is a web application that helps businesses, startups, and freelancers find creative and impactful slogans.  
The user enters their company name and a brief description of their activity, and the AI instantly generates a list of catchy and memorable slogans.  

## 🚀 Features
- AI-powered slogan generation tailored to the business sector.  
- **Free version**: up to **3 slogans/day**.  
- **Premium Subscription 👑**:  
  - Unlimited generations  
  - Save favorite slogans (under development)  
  - Unlimited creativity boost experience

- **Premium**:  (⏳ coming soon) Page dedicated to subscription and exclusive features.
